개인정보처리방침

MediX(이하 "회사")가 운영하는 케미체크(ChemiCheck) 앱(이하 "서비스")은 이용자의 개인정보를 소중히 여기며, 「개인정보 보호법」 및 관련 법령을 준수합니다. 본 방침은 회사가 수집하는 개인정보의 항목, 수집 방법, 이용 목적, 보관 기간, 그리고 이용자의 권리에 대해 안내합니다.

📱

기기 내 저장

가족 프로필·등록 제품은 사용자 기기에만 저장됩니다. 서버 전송 없음.

🔒

사진 미전송

촬영·선택한 사진은 기기 내에서만 처리됩니다. 외부 서버로 전송되지 않습니다.

🤖

AI 상담 텍스트만

AI 상담 입력 텍스트만 Anthropic 서버로 전송됩니다. 가족 정보·사진 미포함.

🗑️

쉬운 삭제

앱 삭제 시 모든 데이터가 완전히 삭제됩니다. 별도 탈퇴 절차 없음.

케미체크는 회원 가입 없이 이용 가능합니다. 이름·이메일·전화번호 등의 식별 정보를 수집하지 않습니다.

제7조 이용자의 권리

이용자는 언제든지 다음 권리를 행사할 수 있습니다:

① 열람 권리

가족 프로필·등록 제품은 앱 내 마이페이지 → 프로필 수정에서 직접 확인할 수 있습니다.

② 수정·정정 권리

마이페이지 → 프로필 수정에서 언제든지 가족 프로필을 수정할 수 있습니다. 등록 제품은 내 제품 화면에서 삭제·관리할 수 있습니다.

③ 삭제 권리

기기에서 앱을 삭제하면 모든 데이터가 즉시 삭제됩니다. AI 상담 기록 삭제를 원하시면 inmani1555@gmail.com으로 문의해 주세요. Anthropic의 데이터 삭제 절차를 안내해 드립니다.

④ 처리 정지 요청

AI 상담 기능을 사용하지 않으면 텍스트가 Anthropic 서버로 전송되지 않습니다. 에어코리아 API 호출을 원하지 않으시면 데모 모드를 활성화하면 오프라인으로 동작합니다.

⑤ 이의 제기

개인정보 관련 불만이 있으시면 개인정보보호위원회(privacy.go.kr) 또는 한국인터넷진흥원(KISA, privacy.kisa.or.kr)에 신고하실 수 있습니다.

제8조 만 14세 미만 아동의 개인정보 보호

케미체크는 만 14세 미만 아동의 개인정보를 별도로 수집하지 않습니다. 만 14세 미만 아동이 서비스를 이용할 경우 보호자의 동의 하에 사용해야 합니다. 만 14세 미만 아동의 개인정보가 수집된 것을 확인하시면 즉시 삭제 요청해 주세요.

제9조 개인정보 보안

  • 기기 내 저장: 가족 프로필 등 민감 정보는 기기 내 UserDefaults에만 저장되며 서버로 전송되지 않습니다.
  • HTTPS 전송: AI 상담·에어코리아 API 등 모든 외부 통신은 TLS(HTTPS)로 암호화됩니다.
  • API 키 보호: Anthropic API 키는 앱 코드에 포함되지 않으며 Cloudflare Workers의 Secret 변수로만 관리됩니다.
  • 온디바이스 OCR: 카메라·갤러리 이미지는 Apple Vision Framework로 기기 내에서만 처리됩니다.

제10조 쿠키 및 추적 기술

케미체크는 쿠키·웹 비콘·광고 추적 SDK를 사용하지 않습니다. App Store 설치 경로 등 Apple이 자체적으로 수집하는 정보는 Apple의 개인정보처리방침을 따릅니다.
